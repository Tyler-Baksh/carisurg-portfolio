# 🏥 CariSurg Portfolio: Application of AI in Caribbean ED Triaging

This repository is a Python-based portfolio focused on using AI to aid in ED triaging in the Caribbean region

It exists to showcase the skills and knowledge acquired over the course of the CariSurg Healthcare AI Training Program. It is intended for anyone interested in AI, medicine and their intersection in the Caribbean.

## 🛠️ Installation

> **Prerequisites:** Python 3.11.9+ and `pip` installed on your machine. It is recommended to use a virtual environment.

1. Clone the repository to your local machine:

```bash
git clone https://github.com/Tyler-Baksh/carisurg-portfolio
```

2. Navigate into the project directory:

```bash
cd carisurg-portfolio
```

3. (Recommended) Create and activate a virtual environment:

```bash
# On Linux/Mac
python -m venv venv
source venv/bin/activate

# On Windows
python -m venv venv
venv\Scripts\activate
```

4. Install all required dependencies:

```bash
pip install -r requirements.txt
```

## 🚀 Usage

### Week 0
Once dependencies are installed, launch Jupyter Notebook to explore the Week 0 notebooks:

```bash
jupyter notebook
```

This will open the Jupyter interface in your browser. Navigate to the `notebooks/` folder and open any `.ipynb` file to run or review an analysis. Each notebook is self-contained and can be executed top-to-bottom with the data files found in the `data/` directory.
