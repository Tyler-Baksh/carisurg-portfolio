# Week 0 Dataset — Orientation & Tool Familiarization

## Overview

This dataset is a reduced, de-identified triage dataset derived from the **Mercer General ED EHR**, used with data governance sign-off from the Mercer Research Ethics Committee.

It contains basic patient demographics alongside common vital signs:

- Pulse
- Temperature
- Respiratory rate
- Blood pressure
- Oxygen saturation

## Purpose

This is an **orientation dataset**. It is intentionally simple and limited in scope.

> **Note:** The Week 5 dataset is the primary research dataset. This Week 0 dataset is for practice only and should not be used for any analytical or clinical conclusions.

## Where This Dataset Belongs

| Property | Detail |
|---|---|
| **Stage** | Week 0 — Onboarding / Tool Familiarization |
| **Source** | Mercer General ED EHR (de-identified) |
| **Governance** | Mercer Research Ethics Committee |
| **Scope** | Demographics + core vital signs |
| **Use** | Practice and orientation only |
| **Not for** | Research analysis or clinical inference |
